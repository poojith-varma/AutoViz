# Deployment

## Frontend

Platform: Vercel

Environment Variables:

```env
VITE_API_URL=https://your-render-url.onrender.com
```

## Backend

Platform: Render

Environment Variables:

```env
GROQ_API_KEY=your_key
```

Build Command:

```bash
npm install && npm run build
```

Start Command:

```bash
npm start
```
