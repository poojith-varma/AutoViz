# 🚀 AutoViz

### AI-Powered Data Cleaning & Analytics Platform

AutoViz is an intelligent analytics platform that automatically cleans, analyzes, visualizes, and explains structured datasets using Artificial Intelligence.

Instead of spending hours manually cleaning spreadsheets, creating dashboards, and searching for insights, users can simply upload a dataset and instantly receive:

* Automated data cleaning
* Dataset quality profiling
* Outlier detection & visualization
* KPI generation
* AI-generated insights
* Natural language dataset chat
* Interactive dashboards
* Exportable analytics

## 🎯 Why AutoViz?

Data analysis often requires multiple tools and repetitive manual work:

* Cleaning missing values
* Removing duplicates
* Detecting anomalies
* Building charts
* Generating reports
* Understanding trends

AutoViz simplifies this workflow into a single intelligent platform.

Upload a dataset and receive meaningful insights within seconds.

## ✨ Key Features

### 🧹 Intelligent Data Cleaning

* Missing value handling
* Duplicate removal
* Empty row detection
* Case normalization
* Whitespace trimming
* Invalid date detection

### 📊 Dataset Profiling

* Dataset overview
* Column classification
* Data quality scoring
* Row and column statistics

### 📈 Automated Dashboard Generation

* AI-recommended charts
* Bar charts
* Line charts
* Pie charts
* Scatter plots

### 🚨 Outlier Intelligence

* IQR-based anomaly detection
* Outlier analysis panel
* Outlier visualization charts

### 🤖 AI Insights

Automatically generated insights about:

* Sales trends
* Customer behavior
* Geographic performance
* Department performance
* Data anomalies

### 💬 AI Dataset Chat

Ask questions about your uploaded dataset using natural language.

Examples:

```text
What is the average sales value?
Which country performs best?
Show profit trends.
```

* Export cleaned CSV
